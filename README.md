# Firebase Backend

## Install

1. Add github npm registry to your .npmrc file
```
registry=https://npm.pkg.github.com/@llewellyncollins
registry=https://registry.npmjs.org
```

2. Install package
```npm install @llewellyncollins/firebase-backend```
